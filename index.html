How Points Work

Our point system is designed to reward real estate professionals for qualifying transactions:

  • Minimum Transaction: Home transactions must be at least $500,000
  • Point Calculation: Every $500,000 in transaction value equals 1 point
  • Rounding System: We round up to the nearest next point for your benefit
  • Example: A $751,000 transaction can register up to 2 points

This generous rounding system ensures you get maximum value from every qualifying transaction.
